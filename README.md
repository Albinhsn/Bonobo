### Bonobo 
A first attempt at writing a language in Haskell. Contains an interpretor, repl, compiler and vm to run the compiled bytecode based on `Writing a compiler in GO` by Thorsten Ball. 

Written as a final project during a 1.5 year trade school education. Knowing nothing about language design, compilers and Haskell for some reason i felt like this was the time to do something about that. If you know any Haskell im sorry, if you decide to look at any of the code I'm sorry, I feel the same pain you do writing this.
